สถานะเครื่อง (จากแอป NKBKConnext)

ข้อมูลจากเซิร์ฟเวอร์ — ใส่ ?key= ใน URL ถ้า Node อยู่ใต้โฟลเดอร์ (เช่น /monitor-api/) ให้กดลิงก์ด้านล่าง หรือใส่ ?api=...
อย่าเปิดจาก file:///

บังคับซิงก์ไป Firebase → คอลเลกชัน programs (หน้าแอดมิน)

ใส่รหัสเดียวกับ MONITOR_SYSTEM_UPLOAD_SECRET บนเซิร์ฟเวอร์ (ในแอป = systemSnapshotUploadSecret ใน monitor-config)
กดปุ่มเพื่อเขียน snapshot ที่เซิร์ฟเวอร์ถืออยู่ตอนนี้ — กด โหลด/รีเฟรช ให้สำเร็จก่อนเพื่อให้รู้ฐาน API
