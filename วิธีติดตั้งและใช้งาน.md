# วิธีติดตั้งและใช้งานบนคอมพิวเตอร์

## สิ่งที่ต้องมีก่อน

ต้องมี **Node.js** ติดตั้งในเครื่องก่อน

### ติดตั้ง Node.js

1. ไปที่ **https://nodejs.org**
2. ดาวน์โหลดตัว **LTS** (ปุ่มสีเขียว)
3. ติดตั้ง แล้วปิด/เปิด Command Prompt หรือรีสตาร์ทเครื่องหนึ่งครั้ง

---

## วิธีเปิดโปรแกรม

### แบบที่ 1 – ดับเบิลคลิก (แนะนำ)

1. เปิดโฟลเดอร์ `C:\Users\PC\Downloads\NKBK\it`
2. **ดับเบิลคลิกไฟล์ `start.bat`**
3. โปรแกรมจะเปิดเป็น**หน้าต่างโปรแกรม** (ไม่ใช่เบราว์เซอร์)
4. เมื่อปิดหน้าต่างหรือกดย่อ โปรแกรมจะ**ย่อไปอยู่ที่แถบด้านล่าง (ถาดระบบ)** ไม่ปิด
5. **ดับเบิลคลิกไอคอนที่ถาดระบบ** เพื่อเปิดหน้าต่างอีกครั้ง
6. คลิกขวาที่ไอคอนที่ถาด: เลือก **ออกจากโปรแกรม** เมื่อต้องการปิดจริง

### แบบที่ 2 – เปิดพร้อม Windows (ให้รันอัตโนมัติเมื่อเปิดเครื่อง)

1. เปิดโปรแกรมตามแบบที่ 1
2. **คลิกขวาที่ไอคอนที่ถาดระบบ** (มุมล่างขวาของจอ)
3. เลือก **เปิดพร้อม Windows** ให้มีเครื่องหมายถูก
4. ตั้งแต่เปิดเครื่องครั้งถัดไป โปรแกรมจะรันอัตโนมัติและย่ออยู่ที่ถาดระบบ
5. ถ้าไม่ต้องการให้รันอัตโนมัติอีก ให้คลิกขวาที่ไอคอนแล้วเอาเครื่องหมาย **เปิดพร้อม Windows** ออก

### แบบที่ 3 – ใช้คำสั่ง

เปิด Command Prompt แล้วรัน:

```text
cd C:\Users\PC\Downloads\NKBK\it
npm start
```

## สรุปการทำงาน

| การกระทำ | ผลลัพธ์ |
|----------|----------|
| ดับเบิลคลิก `start.bat` | เปิดโปรแกรม (หน้าต่าง + ไอคอนที่ถาด) |
| ปิดหน้าต่าง / กดย่อ | โปรแกรมย่อไปอยู่ที่ถาดระบบ (ยังไม่ปิด) |
| ดับเบิลคลิกไอคอนที่ถาด | เปิดหน้าต่างโปรแกรมอีกครั้ง |
| คลิกขวาที่ไอคอนที่ถาด | เมนู: เปิดโปรแกรม / เปิดพร้อม Windows / ออกจากโปรแกรม |
| เลือก "ออกจากโปรแกรม" | ปิดโปรแกรมจริง |

ไม่ต้องเปิดเบราว์เซอร์ แค่เปิดโปรแกรมแล้วดูที่หน้าต่างหรือไอคอนที่แถบด้านล่าง
