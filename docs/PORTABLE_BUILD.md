# แพ็กเก์แบบ Portable (ไม่ใช้ NSIS)

## ต้อง “ติดตั้ง” ลงเครื่องไหม?

**ไม่มีขั้นตอนติดตั้งแบบลงทะเบียน Windows** (ไม่มีรายการใน *การตั้งค่า → แอป → ถอนการติดตั้ง*)

- แค่ **วางไฟล์ `.exe` ไว้ที่ไหนก็ได้** (เช่น Desktop, โฟลเดอร์บริษัท, USB)
- **ดับเบิลคลิก** = รันโปรแกรมทันที  
- โปรแกรมจะสร้างโฟลเดอร์เก็บ **การตั้งค่า / cache** ใน `%APPDATA%` อัตโนมัติ (ไม่ต้องทำอะไรเอง)

สรุป: **ไม่ต้อง “install” แบบเดิม** — แค่มีไฟล์ `.exe` ก็ใช้ได้

- หลังเปิดครั้งแรก (หรือครั้งแรกหลังอัปเดตเวอร์ชันนี้) จะมี **ไอคอนบน Desktop** อัตโนมัติ — ไม่ต้องลากเอง  
- ถ้าไม่ขึ้น: **เมนูถาด → สร้างไอคอนบน Desktop**

---

## ถอนการติดตั้ง / ลบโปรแกรมยังไง?

แบบ Portable **ไม่มีปุ่ม Uninstall ใน Windows** ต้องทำเอง 3 ขั้น:

| ขั้น | ทำอะไร |
|------|--------|
| 1 | **ปิดโปรแกรม** — คลิกขวาไอคอนถาด (มุมล่างขวา) → **ออกจากโปรแกรม** |
| 2 | **ลบไฟล์** `NKBKConnext System … Portable.exe` ที่คุณเคยวางไว้ |
| 3 | *(ถ้าต้องการล้างการล็อกอิน / ค่าที่บันทึกในเครื่อง)* **ลบโฟลเดอร์ข้อมูล** — ปกติอยู่ที่ `%APPDATA%\NKBKConnext System` |

จาก Explorer พิมพ์ `%APPDATA%` ในแถบที่อยู่ แล้วเข้าโฟลเดอร์ชื่อตามแอป (หรือใช้เมนูถาดในโปรแกรม: **วิธีถอนโปรแกรม… → เปิดโฟลเดอร์ข้อมูล**)

## Build

```bash
npm run build:installer
```

หรือ `build-installer.bat` → ได้ไฟล์ **`…Portable.exe`** ใน `dist\`

## อัปเดต

**เมนูถาด → ตรวจสอบอัปเดต** → เปิด GitHub Releases → ดาวน์โหลด `.exe` ใหม่ แทนของเดิม (ปิดโปรแกรมก่อน)

---

## ถ้าต้องการแบบมี Uninstall ใน Windows

ต้องกลับไปใช้ตัวติดตั้งแบบ **NSIS / MSI** (มีรายการใน “แอปที่ติดตั้ง”) — แจ้งทีม dev ถ้าต้องการให้รองรับทั้งสองแบบ
