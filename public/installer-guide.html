NKBKConnext System

คู่มือติดตั้ง — ฟอนต์ Kanit & ธีมเดียวกับโปรแกรม

ขั้นตอนติดตั้ง

  1. ดาวน์โหลดไฟล์ NKBKConnext System Setup … .exe
  2. ดับเบิลคลิกเพื่อเปิด — หน้าต่างติดตั้งเป็นของ Windows (ไม่ใช่หน้าแอป)
  3. กด Next ตามขั้นตอน แล้วเปิดโปรแกรมจาก Desktop หรือ Start Menu

หลังติดตั้ง

หน้าล็อกอินและแดชบอร์ดใช้ฟอนต์ Kanit และดีไซน์ทันสมัยตามที่ออกแบบไว้

ทำไมหน้า Setup ไม่เหมือนแอป? ตัวติดตั้งสร้างด้วยมาตรฐาน NSIS ของ Windows — เปลี่ยนฟอนต์เป็น Kanit ในปุ่มและข้อความของ wizard ไม่ได้ หน้านี้ (เปิดในเบราว์เซอร์) จึงเป็นจุดที่แสดงแบรนด์แบบเต็มรูปแบบได้
