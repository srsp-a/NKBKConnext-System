# วิธีติดตั้งและถอนการติดตั้ง (Install / Uninstall)

## ทำไมต้องใช้ตัวติดตั้ง

ถ้าเปิดโปรแกรมด้วย **start.bat** หรือ **npm start** จะมีหน้าต่าง Command Prompt (หน้าต่างดำ) แสดงขึ้นมา

ถ้าใช้ **ตัวติดตั้ง (Installer)** แทน:
- ติดตั้งครั้งเดียว แล้วเปิดจาก **เมนู Start** หรือ **ไอคอนบน Desktop**
- **ไม่มีการเปิดหน้าต่าง Command Prompt** อีก
- มี **Uninstall** ใน Windows (การตั้งค่า > แอป > แอปที่ติดตั้ง)

---

## ขั้นตอนที่ 1: สร้างตัวติดตั้ง (ทำครั้งเดียว)

1. เปิดโฟลเดอร์โปรเจกต์  
   `C:\Users\PC\Downloads\NKBK\it`
2. **ดับเบิลคลิก `build-installer.bat`**
3. รอจน build เสร็จ (อาจใช้เวลา 2–5 นาที)
4. โฟลเดอร์ **dist** จะเปิดขึ้นมา ข้างในมีไฟล์ **.exe** (ตัวติดตั้ง)

---

## ขั้นตอนที่ 2: ติดตั้งโปรแกรม

1. เปิดโฟลเดอร์ **dist**
2. ดับเบิลคลิกไฟล์ **System Status Monitor 1.0.0 Setup.exe** (หรือชื่อใกล้เคียง)
3. ทำตามขั้นตอนในตัวติดตั้ง:
   - เลือกโฟลเดอร์ติดตั้ง (หรือใช้ค่าที่เสนอไว้)
   - เลือกสร้าง Shortcut บน Desktop / Start Menu ตามต้องการ
   - กด **ติดตั้ง (Install)**
4. ติดตั้งเสร็จแล้ว กด **ปิด (Finish)**

---

## วิธีเปิดโปรแกรมหลังติดตั้ง

- จาก **เมนู Start** พิมพ์ **System Status Monitor** แล้วกดเปิด  
หรือ  
- ดับเบิลคลิก **ไอคอนบน Desktop** (ถ้าเลือกสร้าง Shortcut ตอนติดตั้ง)

**จะไม่มีหน้าต่าง Command Prompt แสดงอีก**

---

## วิธีถอนการติดตั้ง (Uninstall)

1. กด **Win + I** เปิด **การตั้งค่า (Settings)**
2. ไปที่ **แอป** > **แอปที่ติดตั้ง**
3. ค้นหา **System Status Monitor**
4. กด **...** (หรือคลิกที่โปรแกรม) แล้วเลือก **ถอนการติดตั้ง (Uninstall)**
5. ยืนยันตามที่ Windows ถาม

หรือ

1. กด **Win + R** พิมพ์ `appwiz.cpl` กด Enter
2. ในรายการหา **System Status Monitor**
3. คลิกแล้วกด **ถอนการติดตั้ง**

---

## สรุป

| การใช้งาน | วิธี |
|-----------|------|
| สร้างตัวติดตั้ง | ดับเบิลคลิก **build-installer.bat** (ครั้งเดียว) |
| ติดตั้ง | เปิดไฟล์ **.exe** ในโฟลเดอร์ **dist** |
| เปิดโปรแกรม | Start Menu หรือไอคอนบน Desktop (ไม่มีหน้าต่าง CMD) |
| ถอนการติดตั้ง | การตั้งค่า > แอป > แอปที่ติดตั้ง > System Status Monitor > ถอนการติดตั้ง |
